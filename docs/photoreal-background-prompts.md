# Photoreal Background Prompt Pack (8 Scenes)

이 문서는 현재 게임 씬 ID에 맞는 실사형 배경 생성 프롬프트입니다.
출력은 `16:9`, 텍스트/워터마크 없이, 캐릭터 없는 환경 배경으로 고정하세요.

## Global Prompt Rules

- Aspect ratio: `16:9`
- Target size suggestion: `1920x1080` (or `1600x900` for lighter files)
- No characters, no text, no logos, no watermark
- Puzzle readability: center/right area에 장치가 보이도록 구성
- Mood: melancholic retro-industrial, rusted robot city, soft cinematic lighting
- Camera: eye-level wide shot, slight perspective depth

Negative prompt suggestion:
`people, face, human, readable text, logo, watermark, low resolution artifacts, overexposed bloom, cartoon style`
