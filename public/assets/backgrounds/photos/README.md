# Put Photoreal Backgrounds Here

실사 배경 파일을 이 폴더에 넣고 아래 스크립트를 실행하세요.

```bash
npm run backgrounds:apply-photo
```

## Expected base names

- `scrap_yard`
- `sewer_gate`
- `clock_tower`
- `city_gate`
- `steam_square`
- `train_hub`
- `radio_lab`
- `outer_rooftop`

## Supported extensions

- `.webp` (recommended)
- `.png`
- `.jpg`
- `.jpeg`

각 씬당 확장자 하나만 두세요.
예: `scrap_yard.webp`는 가능, `scrap_yard.webp` + `scrap_yard.png` 동시 존재는 불가.
